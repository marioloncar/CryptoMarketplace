# Crypto Marketplace 
Playground project showing crypto market using latest Android technologies.

* MVVM
* Repository pattern
* Modularization 
  
* Kotlin
* Ktor
* Timber
* Koin
* Detekt
* Mockito, JUnit
  
Kotlinx
* DateTime
* Serialization
* Coroutines
  
Jetpack: 
* Lifecycle
* Compose
* Material3
* ViewModel
